## 安裝/使用/操作方式
* **Download**
```gherkin=
git clone https://github.com/harry1003/OnlineShop.git
```
* **Install**
```gherkin=
cd server && npm install
cd client && npm install
```
* **Run**
```gherkin=
cd server && npm start
cd client && npm start
```
* **Our client uses port 3000, and server uses port 3001. Please make sure these ports are available.**
